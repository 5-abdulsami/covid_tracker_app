# Covid 19 Tracker App

## Overview
The Covid 19 Tracker App is a Flutter application designed to provide users with up-to-date information on Covid-19 statistics worldwide. Users can view global statistics as well as detailed statistics for individual countries. Additionally, the app offers a search feature allowing users to find specific countries and retrieve their Covid-19 statistics.

## Features
1. Global Stats: View current Covid-19 statistics worldwide.
2. Country Stats: Explore detailed Covid-19 statistics for individual countries.
3. Search Functionality: Search for a specific country to access its Covid-19 statistics easily.
4. 
## Usage
1. Upon opening the app, a Splash Screen is displayed following the global Covid-19 statistics.
2. To view statistics for a specific country, either scroll through the list or use the search functionality.
3. Tap on a country to view its detailed Covid-19 statistics.
4. To search for a specific country, use the search bar at the top of the screen and enter the country's name.
5. Enjoy staying informed about Covid-19 statistics worldwide!

## Disclaimer
The Covid 19 Tracker App is intended for informational purposes only. While every effort has been made to ensure the accuracy of the data provided, users should refer to official sources for the most up-to-date information on Covid-19.
